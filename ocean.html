Sobre o projeto

Utilizamos o Visual Studio Community, para criar um projeto Aplicativo de Console (console app) como nome ProjetoElevador, utilizando a estrutura .NET 5. Todas as classes estão em uma pasta chamada Model.

Criamos uma classe denominada Elevador para armazenar as informações de um elevador dentro de um prédio, a classe armazena o andar atual (térreo = 0), total de andares no prédio, capacidade do elevador e quantas pessoas estão presentes nele.

A classe também disponibiliza os seguintes métodos:


1º Inicializar: que recebe como parâmetros a capacidade do elevador e o total de andares no prédio (os elevadores sempre começam no térreo e vazio);

2º Entrar: que acrescenta uma pessoa no elevador (só acrescenta se ainda houver espaço);

3º Sair: que remove uma pessoa do elevador (só remove se houver alguém dentro dele);

4º Subir: que sobe um andar (não sobe se já estiver no último andar);

5º Descer: que desce um andar (não desce se já estiver no térreo);

Como utilizar o simulador

O Sistema irá pedir ao usuário para informar a capcaidade máxima de pessoas que o elevador suporta, logo após, ele também irá informar número andares que o prédio possui.

Em sequência, irá aparecer um painel de opções onde o usuário poderá interagir com o simulador.
